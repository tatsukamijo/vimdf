Appearance

Navigation

Page scroll aliases

Bind an extra key (with or without modifiers) to each page-scroll command. Focus a field and press the combo you want, or press Backspace to clear. Ctrl-d / Ctrl-u / Ctrl-f / Ctrl-b always remain active. Note: f is used by link hints, so avoid plain f for full-page down.

Caret & selection

Color used for the caret, visual selection, new highlights, and the status-bar mode label (NORMAL, VISUAL, etc). Existing highlights keep their original color.

Link hints

Colors for the yellow hint labels shown when pressing f. Click Reset to restore the built-in color.

Status bar

Colors fall back to the current theme when unset.

State persistence

Local file access

To open file:// PDFs automatically, visit chrome://extensions, find VimDF, and enable Allow access to file URLs.

Keybindings

Press ? inside the viewer to see the full list. Custom keymap editing is coming in a future version.
